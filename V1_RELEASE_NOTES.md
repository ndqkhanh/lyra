## 🚀 GETTING STARTED

### Installation

```bash
# From PyPI (when released)
pip install lyra-cli

# Or from source
git clone https://github.com/ndqkhanh/lyra
cd lyra
pip install -e .
```

### Quick Start

```bash
# Start Lyra
lyra

# Research something
> /research quantum computing

# Switch model
> /model claude-opus-4.7

# Start a team
> /team run "Build a REST API"

# Store a lesson
> /reflect tag:testing verdict:success :: Always write tests first

# Check status
> /status
```
