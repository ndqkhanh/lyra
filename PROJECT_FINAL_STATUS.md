## 🚀 QUICK START

### Use Lyra Now:
```bash
# Install
cd projects/lyra/packages/lyra-cli
pip install -e .

# Run
lyra

# Credentials auto-loaded from:
# - ~/.claude/settings.json (Claude Code)
# - ~/.lyra/credentials.json (Lyra)
```

### Available Commands:
