# Debugging Techniques

## Systematic Approach

1. **Reproduce**: Consistently reproduce the bug
2. **Isolate**: Narrow down the problem area
3. **Hypothesize**: Form theories about the cause
4. **Test**: Verify hypotheses
5. **Fix**: Apply the solution
6. **Verify**: Confirm the fix works

## Tools

- **Debugger**: Step through code, inspect variables
- **Logging**: Add strategic log statements
- **Profiler**: Identify performance bottlenecks
- **Network Inspector**: Debug API calls
- **Stack Traces**: Understand error flow

## Common Techniques

- Binary search (comment out half the code)
- Rubber duck debugging
- Print debugging
- Unit test isolation
- Git bisect for regressions
