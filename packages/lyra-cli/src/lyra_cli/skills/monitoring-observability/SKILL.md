# Monitoring & Observability

## Three Pillars

### Metrics
- Response times
- Error rates
- Resource usage
- Business metrics

### Logs
- Structured logging
- Log levels (DEBUG, INFO, WARN, ERROR)
- Correlation IDs
- Centralized logging

### Traces
- Distributed tracing
- Request flow
- Performance bottlenecks
- Service dependencies

## Best Practices

- Monitor what matters
- Set up alerts
- Use dashboards
- Track SLOs/SLIs
- Regular reviews

## Tools

- Prometheus (metrics)
- Grafana (visualization)
- ELK Stack (logs)
- Jaeger (tracing)
