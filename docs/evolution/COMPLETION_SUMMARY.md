### 1. Temporal Memory
- Facts have validity windows (valid_from/valid_until)
- Automatic superseding of outdated information
- Temporal query filtering

### 2. Hybrid Retrieval
- BM25 keyword search + vector semantic search
- Configurable weighting (hybrid_alpha)

### 3. Verifier-Gated Everything
- Memory writes verified before storage
